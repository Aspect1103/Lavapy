Lavapy
========


A modern, easy to use, feature-rich, and async ready API wrapper for Discord written in Python.

Installation
---------------------------
Lavapy requires Python 3.8 or higher

.. code:: sh

    # Windows
    py -3.8 -m pip install -U lavapy

    # Linux/macOS
    python3.8 -m pip install -U lavapy
